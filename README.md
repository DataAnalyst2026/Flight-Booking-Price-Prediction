# ✈️ Flight Booking Price Prediction using Machine Learning

## 📖 Project Overview

This project predicts flight ticket prices using machine learning techniques. The objective is to analyze historical flight booking data, identify the factors that influence ticket prices, and build a predictive model capable of estimating flight fares accurately.

---

## 🎯 Objectives

- Analyze flight booking data.
- Perform data cleaning and preprocessing.
- Explore relationships between different flight features and ticket prices.
- Build and evaluate machine learning models for price prediction.
- Compare model performance to identify the best model.

---

## 📂 Dataset Features

The dataset includes the following features:

- Airline
- Date of Journey
- Source
- Destination
- Route
- Departure Time
- Arrival Time
- Duration
- Total Stops
- Additional Information
- Price (Target Variable)

## 🤖 Machine Learning Models

- Linear Regression
- Decision Tree Regressor

---

## 📈 Results

The project compares multiple regression models and selects the one with the best predictive performance based on evaluation metrics such as R² Score, MAE, RMSE, and MSE.
